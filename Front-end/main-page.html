Ταξιδεύοντας με κατοικίδια

Τα καλύτερα ταξίδια γίνονται με τους καλύτερους φίλους!!!

Αν ετοιμάζεστε για ταξίδι, δεν υπάρχει κανένας λόγος να μην πάρετε μαζί σας τα κατοικίδιό σας. Ενημερωθείτε για τους όρους και τις προϋποθέσεις κάθε αεροπορικής εταιρείας, ενημερώστε τη για τις πρόθεσή σας να συνταξιδέψετε με τον μικρό σας φίλο και είστε έτοιμοι για απογείωση.

Συναλλαγματικοί περιορισμοί

Όσα πρέπει να ξέρετε για το συνάλλαγμα

Κανόνες για την κατοχή μετρητών και ισοδυνάμων αυτών σε ταξίδια εντός της Ευρωπαϊκής Ένωσης:

Όταν ταξιδεύετε μεταξύ δύο χωρών - μελών της Ευρωπαϊκής Ένωσης, είστε υποχρεωμένοι να δηλώσετε προφορικά στους τελωνειακούς υπαλλήλους εάν έχετε μαζί σας χρηματικό ποσό(μετρητά ή ισοδύναμα) αξίας που ισούται ή υπερβαίνει τα 10.000€.
Τα ακόλουθα νομισματικά μέσα υπόκεινται σε αυτή την απαίτηση:
  • Μετρητά χρήματα(νομίσματα και χαρτονομίσματα)
  • Κινητές Αξίες
  • Πολύτιμοι Λίθοι
  • Πολύτιμα Μέταλλα
Λεπτομερείς πληροφορίες σχετικά με την υποχρέωσή σας να δηλώσετε μετρητά και ισοδύναμα μετρητών όταν ταξιδεύετε εντός της Ευρωπαϊκής Ένωσης δείτε εδώ

Όταν ταξιδεύετε προς ή από χώρες εκτός Ευρωπαϊκής Ένωσης:

Εάν ταξιδεύετε από μία χώρα της ΕΕ προς μία χώρα εκτός ΕΕ ή αντιστρόφως, θα πρέπει να δηλώνετε στις τελωνειακές Αρχές τα μετρητά ή τα ισοδύναμα μετρητών αξίας 10.000 ευρώ ή μεγαλύτερης. Η δήλωση αυτή πρέπει να γίνει εγγράφως στην αρμόδια τελωνειακή υπηρεσία.
Κατά την άφιξή σας από χώρα εκτός ΕΕ, θα πρέπει να υποβάλετε στην γραπτή σας δήλωση στις τελωνειακές Αρχές του πρώτου σημείου ελέγχου. Κατά την αναχώρησή σας από χώρα εκτός ΕΕ, σας παρακαλούμε να αναζητήσετε τις πινακίδες που θα σας οδηγήσουν στην αρμόδια τελωνειακή υπηρεσία ή να ρωτήσετε σε ποια θυρίδα ελέγχου μπορείτε να υποβάλετε τη γραπτή σας δήλωση. Θα πρέπει να έχετε υποβάλει τη δήλωση πριν φθάσετε στο σημείο ελέγχου ασφαλείας.
Τα νομισματικά μέσα που θα πρέπει να δηλώνονται είναι τα εξής:
  • Μετρητά
  • Βιβλιάρια τραπέζης
  • Επιταγές
  • Εντολές πληρωμής
  • Συναλλαγματικές
  • Γραμμάτια σε διαταγή
  • Μετοχές
  • Ομόλογα
  • Κουπόνια τοκομεριδίων

Σημεία ελέγου

Προετοιμαστείτε σωστά

Σημαντικές πληροφορίες για τις διαδικασίες ελέγχου ασφαλείας και τα αντικείμετα τα οποία δεν επιτρέπεται να έχετε μαζί σας.

  • Στο σημείο ελέγχου, εσείς και τα πράγματά σας ελέγχονται προκειμένου να διασφαλιστεί ότι δεν έχετε μαζί σας κάποιο αντικείμενο το οποίο απαγορεύεται, βάσει των κανονισμών της ΕΕ ή του ΙΑΤΑ. Για το λόγο αυτό, η χειραποσκευή και το πανωφόρι σας περνούν από ακτίνες Χ.
  • Όσο οι χειραποσκευές και το πανωφόρι σας ελέγχονται με ακτίνες Χ, εσείς θα πρέπει να περάσετε μέσα από τον ανιχνευτή μετάλλων. Για το λόγο αυτό, αφήνετε μικροαντικείμενα όπως κλειδιά, κινητό τηλέφωνο, κέρματα και ταξιδιωτικά έγγραφα, στις τσέπες του παλτού ή του σακακιού σας.
  • Εάν στο σώμα σας υπάρχει βηματοδότης, μόσχευμα καρδιοπνευμονικής παράκαμψης, καθετήρας ή κοχλιακό μόσχευμα, σας παρακαλούμε να ενημερώσετε κάποιο μέλος του προσωπικού ασφαλείας πριν περάσετε από τον ανιχνευτή μετάλλων.

Υγρά και Gel στη χειραποσκευή

  • Πριν φύγετε από το σπίτι σας, προσπαθήστε να τοποθετήσετε όλα τα δοχεία που περιέχουν υγρά ή gel στις αποσκευές τις οποίες θα παραδώσετε. Πάρτε μαζί σας στη χειραποσκευή μόνον τα πιο απαραίτητα αντικείμενα.
  • Σας παρακαλούμε να συμμορφώνεστε με τους κανόνες για τα υργά και τα προϊόντα τύπου gel που μπορείτε να έχετε στη χειραποσκευή σας.
  • Τα δοχεία της χειραποσκευής που περιέχουν υγρά ή gel δεν θα πρέπει να έχουν χωρητικότητα μεγαλύτερη των 100ml το καθένα. Θα πρέπει να τοποθετούνται σε διαφανή, επανασφραγιζόμενη πλαστική σακούλα, η μέγιστη χωρητικότητα της οποίας δεν θα υπερβαίνει το ένα λίτρο-για παράδειγμα, σε μία κανονική σακούλα αποθήκευσης με φερμουάρ. Κάθε επιβάτης επιτρέπεται να έχει μαζί του μόνο μία τέτοια διαφανή σακούλα του ενός λίτρου, στην οποία θα περιέχονται δοχεία με υγρά και/ή άλλα αντικείμενα σε μορφή gel.
  • Δεν υπάρχει περιορισμός στα φάρμακα και τις ειδικές τροφές που είναι απαραίτητες για ιατρικούς λόγους(π.χ. βρεφικές τροφές ή συγκεκριμένα φαγητά λόγω εξειδικευμένων διαιτητικών αναγκών) τα οποία θα χρειαστείτε κατά τη διάρκεια της πτήσης και/ή θα ήταν δύσκολο ή αδύνατο να προμηθευτείτε στην πόλη του τελικού σας προορισμού.

Είδη απο τα καταστήματα του αεροδρομίου

Τα είδη που αγοράσατε από τα καταστήματα του αεροδρομίου ή κατά τη διάρκεια της πτήσης σας, μπορείτε να τα έχετε μαζί σας και μετά το σημείου ελέγχου χειραποσκευών, υπό την προυπόθεση ότι αυτά βρίσκονται τοποθετημένα σε ειδική διαφανή σακούλα η οποία θα έχει σφραγιστεί από τον υπάλληλο στο σημείο πώλησης. Η απόδειξη αγοράς στην οποία αναγράφεται το είδος και το σημείο πώλησης θα πρέπει να είναι επικολλημένη σε εμφανές σημείο, στο εξωτερικό της σακούλας.
